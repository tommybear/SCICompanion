=================================
Sounds (of :class:`EventHandler`)
=================================

.. class:: Sounds

	Defined in Game.sc.

	
	This is a static class used to contain all sounds in the game.
	It is accessed through gSounds and allows pausing or unpausing all game sounds at once.
	
	Example usage::
	
		(gSounds pause: TRUE)
