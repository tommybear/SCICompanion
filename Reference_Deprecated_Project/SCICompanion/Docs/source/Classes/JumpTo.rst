=========================
JumpTo (of :class:`Jump`)
=========================

.. class:: JumpTo

	Defined in Jump.sc.

	
	This makes an :class:`Actor` jump to a certain location.
	
	Example usage::
	
		(theRobot setMotion: JumpTo 240 150)
