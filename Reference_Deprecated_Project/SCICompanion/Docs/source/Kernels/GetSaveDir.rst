===================
GetSaveDir (Kernel)
===================

.. function:: GetSaveDir()

	
	Retrieves a pointer to the string containing the save directory and returns it.
	
	:returns: A pointer to a string containing the save directory.
	
	Example::
	
		(= theString (GetSaveDir))
		(Print theString)
